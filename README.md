# Sneaker-Sales-Analytics
A comprehensive analysis of sneaker sales data using Python and machine learning techniques. Includes visualizations, brand/gender trends, and predictive modeling for sold prices. Dashboard integration via Tableau for business insights.


## 🔍 Project Overview

This project answers key business questions:
- What are the top brands by gender category?
- How do prices fluctuate over time?
- Do collaborations impact sold price?
- What features most influence the sold price?
- Can we predict sold price using regression models?

## 📊 Key Visualizations

- Trend of shoe sales by gender (monthly)
- Top brands sold by gender
- Average sold price of top brands over time
- Price distribution across brands and sources
- Feature correlation heatmap
- Tableau dashboard for pricing and sales performance

## ⚙️ ML Modeling

Models Used:
- Linear Regression
- Lasso Regression
- Ridge Regression
- Random Forest Regressor

Best R² Score: **0.39** (Random Forest)  
Most important feature: **Silhouette**

## 📈 Dashboard Insights (Tableau)

Dashboard Highlights:
- Sold Price Forecast till 2026
- Retail vs Sold Price comparison
- Retailer-wise total sales
- Top collaborators and silhouettes
- YOY Sales comparison
